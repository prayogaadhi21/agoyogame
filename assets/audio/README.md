# Audio Directory

Place your audio files here. All files are optional.

| Filename           | Type | Usage                     |
|:-------------------|:-----|:--------------------------|
| jump.wav           | SFX  | Player flap sound         |
| point.wav          | SFX  | Score increment sound     |
| hit.wav            | SFX  | Collision / hit sound     |
| gameover.wav       | SFX  | Game over sting           |
| background.mp3     | Music| Looping background music  |

Missing audio files are handled gracefully — the game runs without them.
